Converting WebLS Logicbases

Tools | Convert Knowledgebase

The Workshop has the ability to convert many WebLS logicbases to KnowledgeWright knowledgebases. If you do not use modules in your logicbase, then the convert command should work without modification. If you have modules, you have two choices:

  • Convert each module into a separate knowledgebase. In this case, you need to edit each of your module files to include a 'main' module, with all the appropriate attributes.
  • Merge all the modules into one knowledgebase. In this case, you need to rework all your module definitions into one 'main' module.

The convert command will create a new text object called 'main_doc' that incorporates all your headers, footers and goal list into a single output document. As such, 'main_doc' becomes the single goal of the new knowledgebase. Rules are grouped together by goal and one rule_set object is created for each goal. Answers are converted into text objects.

Running on the Web
