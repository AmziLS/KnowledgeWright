Common Workshop Problems

Asian Users

Font specified in font.properties not found [Font_Name]

In the knowledgewright4/workshop/jre/lib subdirectory are a set of files named font.properties.CC where CC is the ISO country code (i.e. ja for Japan, th for Thailand). The top section of this file defines the local font to be used for the default fonts in Java (dialog, dialoginput, serif, sansserif and monospaced). You will get the above error message when the font specified in this file is not installed on your computer.

To fix the problem, edit the appropriate font.properties.CC file and change the name of the missing font to an installed font. Detailed documentation can be found at http://java.sun.com/j2se/1.3/docs/guide/intl/fontprop.html
