Install & Un-Install Instructions

Installation

Windows

  1. Download the latest KnowledgeWright executable file from www.amzi.com/download.
  2. Run kwNNN_4-X-X_win.exe to install the product. This will create a program group for KnowledgeWright with icons for the Workshop and Documentation.
  3. Open the knowledgebase samples/basic/tutorial/shapes.kb. Look at the 'rule_set's. Then try Run/Run.
  4. Open the Documentation. We recommend that you read the Overview and do the Tutorial.

Linux

  1. Download the latest KnowledgeWright file from www.amzi.com/download.
  2. Unzip the distribution file into a directory. Use a tar/gunzip program that restores the full knowledgewright directory tree.
  3. Test the workshop by running kw in the workshop directory. Open the knowledgebase samples/basic/tutorial/shapes.kb. Look at the 'rule_set's. Then try Run/Run.
  4. Open the docs/knowledgewright_manual.html document. We recommend that you read the Overview and do the Tutorial.

Un-Install and Repair

Windows

  1. To remove KnowledgeWright
    In your Control Panel, select Add/Remove Programs. Click on "Remove" to remove it.
  2. To repair KnowledgeWright
    In your Control Panel, select Add/Remove Programs. Click on "Change" or "Repair" to repair it.

Linux

  1. Delete the KnowledgeWright directory tree, knowledgewright_4-X-X.
