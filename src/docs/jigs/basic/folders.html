Folders

Description

Folders let you organize your objects into a meaningful hierarchy. The top level folder, called the root, is designated by /. By default all new objects are put in the root. You can create any number of subfolders and move your objects into them.

Naming Restrictions

Folder names can be comprised of any characters or letters, except they may not contain a forward slash, /.
