Basic Jig Tutorial

The KnowledgeWright Basic Jig supports general-purpose knowledgebases that can be used for a wide variety of advice-giving expert systems. Basic Jig knowledgebases can be deployed either as embedded components of a larger application, or directly on a Web server.

  • Introduction - Goal-seeking expert systems
  • Facts & Goals - The "hello" application
  • Dynamic Text - Smarter "hello"
  • Rules - Diagnosing a problem
  • Chaining Rules - Recommending actions for the problem
  • Creating Reports - Reporting on problem and action
  • Multiple Values - Solving multiple problems
  • Numbers & Formulas - Playing with shapes
  • Dates - How to use dates and perform arithmetic on them.
  • Rules_Tables - Using rules_tables in place of rule_sets
  • Data_Table Queries - Getting data from data tables in the knowledgebase.
  • Database Queries - Using facts from external databases
  • Writing Knowledgebases for the Web - Tips and hints for deploying web-based knowledgebases
