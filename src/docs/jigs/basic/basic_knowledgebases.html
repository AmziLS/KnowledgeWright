Basic Knowledgebases

Description

A knowledgebase consists of folders and objects. The folders organize the objects into a meaningful tree. You can create as many, or as few, folders as you need. The object types are:

  • knowledgebase
  • fact
  • question
  • sql
  • rule_set
  • rules_table
  • data_table
  • text

Each object (except a knowledgebase and data_table) represent a value. So anywhere a value is needed, you can use the name of an object instead. In this case, the KnowledgeWright reasoning engine determines a value for that object. Values can also be combined into expressions.
