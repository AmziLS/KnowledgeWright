Creating Reports

Reporting on problem and action

samples/basic/tutorial/diagnose_3.kb

In the section on dynamic text we saw how a text document can contain subgoals. We can use that feature to generate a report for the user of our diagnostic system.

  1. Create a new text object, named problem_report.
  2. Add text of the form:
    The problem you are having is %problem%. We recommend you %action%. Good luck.
  3. Change the goal of the knowledgebase to problem_report.
  4. Run and experiment.
