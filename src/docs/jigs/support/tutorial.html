Support Jig Tutorial

The KnowledgeWright Support Jig supports general-purpose knowledgebases that can be used for a wide variety of advice-giving expert systems. Support Jig knowledgebases can be deployed either as embedded components of a larger application, or directly on a Web server. The Support Jig is a superset of the Basic Jig. We recommend you run through the Basic Jig Tutorial first.

  • Introduction - Specialized goal seeking expert system
  • Solutions - Create a single-solution knowledgebase
  • Default Solutions - Displaying a response when no solution is found
  • External_Actions - E-mailing the problem to support personnel
  • Additional Solutions - Adding more solutions
